<a id="Introdução"></a>
## 🧩 Introdução 

  ***⠀⠀⠀⠀Esta aplicação é um clone personalizado do Tinder que permite ao usuário escolher entre dar match, recusar ou resetar. Utilizando a biblioteca axios, a aplicação tem acesso à API que contém um banco de dados com cadastros de todas as pessoas. Além disso, a comunicação entre o front-end e o back-end é em tempo real, garantindo uma experiência mais fluida para o usuário.***

## `📖 Instalação` 
  
  
 <br /> 

> Caso tenha Git basta da git clone, caso não tenha baixe o projeto completo em dowlon

```BASH
git clone https://github.com/MayconCoutinho/Astromatch
```

<br /> 

> Caso já tenha o Node em sua maquina basta instalar o projeto com npm i

```BASH
npm i 
```
